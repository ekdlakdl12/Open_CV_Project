## ✨ 주요 기능 (Key Features)

### 1️⃣ 번호판 인식 및 위반 감시
* **LPR(License Plate Recognition):** 저해상도 영상에서도 고신뢰성 텍스트 추출
* **속도 측정:** 실시간 객체 추적을 통한 법규 위반(과속/저속) 자동 판독

### 2️⃣ 차선 분석 및 기록
* **Lane Detection:** 버스 전용차로 위반 및 이탈 실시간 감지
* **Auto Recording:** 위반 상황 발생 시 증거 영상 자동 저장 및 DB 기록
